# ContentCrafter
A full-stack AI-powered content generation platform that combines chat-based AI assistance, genre-based content management, and real-time trending topic discovery from LinkedIn and Twitter.

#✨ Features

# # Module 1: Chat-Based Content Generation
1. Real-time AI writing assistance using Stream Chat

2. Interactive chat interface with AI responses

3. Writing prompts and suggestions

4. Message persistence and history

## Module 2: Genre-Based Content Discovery
1. Browse content by multiple genres

2. Advanced search and filtering

3. Full content display view

4. Content recommendations


## Module 3: Trendy Topics Discovery
1. Real-time trending topics from LinkedIn and Twitter

2. Automatic content generation based on trends

3. Topic ranking and filtering

4. Integration with AI content generator


## Core Features
1. Firebase Authentication (Login/Signup)

2. Secure user sessions and token management

3. Real-time AI content generation via Gemini API

4. Stream Chat integration for live messaging

5. Responsive dark-themed UI

6. Error handling and status indicators


## 🛠️ Tech Stack

## Frontend
1. React 18+

2. Vite - Build tool

3. Tailwind CSS - Styling

## Backend
1. Node.js - Runtime

2. Express.js - Framework

3. Firebase - Authentication & Database

4. Open AI API - Content generation

5. Stream Chat API - Real-time messaging

6. MongoDB - Additional data storage

External APIs
Open AI - Content generation

Stream Chat - Real-time messaging

Firebase Authentication - User auth

## 🚀 Installation

Prerequisites

1. Node.js 16+

2. npm or yarn

3. Firebase account
4. OpenAI API key
5. Stream Chat API keys
6. LinkedIn & Twitter API credentials
